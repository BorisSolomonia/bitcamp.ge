1. Promise Basics: Create a promise that resolves after 2 seconds and logs a success message.
1. Promise Chain: Create a chain of promises where each resolves after 1 second and logs a message.
1. Async/Await: Write an asynchronous function using async/await that simulates fetching data from an API after 2 seconds.
1. Error Handling: Modify the previous example to include error handling using try/catch.
1. Fetch API: Use the Fetch API to make a simple GET request to a mock API and log the response.
1. Async Function with Fetch: Create an asynchronous function that uses the Fetch API to get data from an API and log it.
