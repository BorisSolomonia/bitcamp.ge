## Python - Backend (Django & Django Rest Framework)

Python-рЃўрЃА рЃЉрЃћрЃЦрЃћрЃюрЃЊ рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃњрЃќрЃљ рЃњрЃљрЃўрЃЎрЃЋрЃџрЃўрЃЮ рЃЋрЃћрЃЉ рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃЏрЃћрЃюрЃбрЃерЃў рЃЊрЃљ рЃерЃћрЃЦрЃЏрЃюрЃљ рЃЊрЃўрЃюрЃљрЃЏрЃўрЃБрЃарЃў, рЃЏрЃљрЃдрЃљрЃџрЃў рЃЊрЃЮрЃюрЃўрЃА рЃЋрЃћрЃЉ рЃљрЃърЃџрЃўрЃЎрЃљрЃфрЃўрЃћрЃЉрЃў. 

### рЃЋрЃўрЃАрЃЌрЃЋрЃўрЃАрЃљрЃљ рЃћрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ?
рЃћрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљ рЃћрЃарЃњрЃћрЃЉрЃљ рЃАрЃарЃБрЃџрЃўрЃљрЃЊ 0 - рЃўрЃЊрЃљрЃю рЃЊрЃљрЃЏрЃгрЃДрЃћрЃЉрЃћрЃЉрЃА ­ЪћЦ

### рЃЋрЃўрЃю рЃўрЃДрЃћрЃюрЃћрЃЉрЃА Pythons - рЃА рЃЊрЃљ Django - рЃА?
рЃўрЃАрЃћрЃЌрЃў рЃњрЃўрЃњрЃљрЃюрЃбрЃў рЃЎрЃЮрЃЏрЃърЃљрЃюрЃўрЃћрЃЉрЃўрЃА рЃърЃџрЃљрЃбрЃцрЃЮрЃарЃЏрЃћрЃЉрЃў, рЃарЃЮрЃњрЃЮрЃарЃўрЃфрЃљрЃљ Spotify, Instagram, Dropbox, Pinterest, Youtube, Nasa, National Geographic рЃЊрЃљ рЃАрЃ«рЃЋрЃљ рЃЏрЃарЃљрЃЋрЃљрЃџрЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃљ рЃерЃћрЃЦрЃЏрЃюрЃўрЃџрЃўрЃљ Python - рЃўрЃАрЃљ рЃЊрЃљ Django - рЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃўрЃЌ.

### рЃарЃЮрЃњрЃЮрЃарЃўрЃљ рЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃў Python - рЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃљрЃќрЃћ?
- р▓ърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў рЃЊрЃљрЃћрЃБрЃцрЃџрЃћрЃЉрЃў Back-End рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃА рЃерЃћрЃЦрЃЏрЃюрЃўрЃА рЃ«рЃћрЃџрЃЮрЃЋрЃюрЃћрЃЉрЃљрЃА, RESTful API-рЃћрЃЉрЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃљрЃА рЃЊрЃљ р▓ЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃљрЃарЃЕрЃћрЃЋрЃўрЃЌрЃљрЃЊ рЃерЃћрЃњрЃўрЃФрЃџрЃўрЃљ рЃўрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃАрЃ«рЃЋрЃљ Python-рЃўрЃА рЃцрЃарЃћрЃўрЃЏрЃЋрЃЮрЃарЃЎрЃћрЃЉрЃў - Flask рЃЊрЃљ FastAPI. 
- рЃљрЃЏ рЃДрЃЋрЃћрЃџрЃљрЃцрЃћрЃарЃЌрЃљрЃю рЃћрЃарЃЌрЃљрЃЊ рЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ рЃАрЃ«рЃЋрЃљрЃЊрЃљрЃАрЃ«рЃЋрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃћрЃЉрЃўрЃА рЃњрЃљрЃЏрЃЮрЃДрЃћрЃюрЃћрЃЉрЃљрЃА рЃЊрЃљ рЃЏрЃљрЃЌрЃў рЃЊрЃљрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃЌ, рЃЎрЃЮрЃЏрЃърЃџрЃћрЃЦрЃБрЃарЃў рЃАрЃўрЃАрЃбрЃћрЃЏрЃћрЃЉрЃўрЃА рЃўрЃЏрЃърЃџрЃћрЃЏрЃћрЃюрЃбрЃљрЃфрЃўрЃљрЃА. 
- рЃЏрЃўрЃўрЃдрЃћрЃЉ рЃЏрЃЮрЃюрЃљрЃгрЃўрЃџрЃћрЃЮрЃЉрЃА рЃБрЃљрЃЏрЃарЃљрЃЋ рЃърЃарЃЮрЃћрЃЦрЃбрЃерЃў, рЃњрЃљрЃўрЃфрЃюрЃЮрЃЉ рЃАрЃ«рЃЋрЃљ рЃАрЃбрЃБрЃЊрЃћрЃюрЃбрЃћрЃЉрЃА рЃЊрЃљ рЃњрЃљрЃљрЃЎрЃЌрЃћрЃЉ рЃЏрЃљрЃЌрЃЌрЃљрЃю рЃЎрЃЮрЃџрЃљрЃЉрЃЮрЃарЃљрЃфрЃўрЃБрЃџ рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃА.
- рЃЕрЃЋрЃћрЃюрЃЌрЃљрЃю рЃАрЃгрЃљрЃЋрЃџрЃўрЃА рЃърЃарЃЮрЃфрЃћрЃАрЃерЃў рЃерЃћрЃЦрЃЏрЃюрЃў рЃАрЃљрЃЎрЃЏрЃљрЃЮрЃЊ рЃЊрЃўрЃЊрЃў рЃЏрЃЮрЃфрЃБрЃџрЃЮрЃЉрЃўрЃА рЃарЃћрЃќрЃўрЃБрЃЏрЃћрЃАрЃљ рЃЊрЃљ рЃърЃЮрЃарЃбрЃцрЃЮрЃџрЃўрЃЮрЃА.


- р▓АрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃърЃарЃЮрЃфрЃћрЃАрЃў рЃЏрЃЮрЃўрЃфрЃљрЃЋрЃА рЃЎрЃЋрЃўрЃарЃљрЃерЃў 1 рЃЌрЃћрЃЮрЃарЃўрЃБрЃџ рЃЊрЃљ 3 рЃърЃарЃљрЃЦрЃбрЃўрЃЎрЃБрЃџ (рЃАрЃћрЃЏрЃўрЃюрЃљрЃарЃў) рЃџрЃћрЃЦрЃфрЃўрЃљрЃА.
- рЃЌрЃБ рЃАрЃљрЃЏрЃћрЃюрЃбрЃЮрЃарЃЮ рЃърЃљрЃЎрЃћрЃбрЃА рЃерЃћрЃўрЃФрЃћрЃю,  рЃЎрЃЋрЃўрЃарЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў, рЃАрЃљрЃдрЃљрЃЏрЃЮрЃА рЃАрЃљрЃљрЃЌрЃћрЃЉрЃерЃў, рЃЎрЃЮрЃюрЃбрЃљрЃЦрЃбрЃў рЃњрЃћрЃЦрЃюрЃћрЃЉрЃљ, рЃњрЃљрЃЏрЃЮрЃфрЃЊрЃўрЃџ рЃЏрЃћрЃюрЃбрЃЮрЃарЃћрЃЉрЃЌрЃљрЃю, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃдрЃћрЃерЃў 4 рЃАрЃљрЃљрЃЌрЃўрЃА рЃњрЃљрЃюрЃЏрЃљрЃЋрЃџрЃЮрЃЉрЃљрЃерЃў, рЃАрЃљрЃдрЃљрЃЏрЃЮрЃА 18:00-рЃўрЃЊрЃљрЃю 22:00-рЃЏрЃЊрЃћ рЃърЃўрЃарЃљрЃЊрЃљрЃЊ рЃерЃћрЃю рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃљрЃю рЃДрЃЋрЃћрЃџрЃљрЃюрЃљрЃўрЃарЃў рЃърЃарЃЮрЃЉрЃџрЃћрЃЏрЃўрЃА рЃњрЃљрЃЊрЃљрЃГрЃарЃљрЃерЃў. р▓ўрЃАрЃўрЃюрЃў рЃЏрЃЮрЃњрЃфрЃћрЃЏрЃћрЃю рЃарЃЕрЃћрЃЋрЃћрЃЉрЃА рЃЊрЃљ рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃўрЃљрЃю рЃърЃарЃЮрЃћрЃЦрЃбрЃћрЃЉрЃўрЃА рЃљрЃгрЃДрЃЮрЃЉрЃљрЃерЃў.
- р▓ЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃњрЃћрЃЦрЃюрЃћрЃЉрЃљ рЃгрЃЋрЃЊрЃЮрЃЏрЃљ рЃљрЃџрЃњрЃЮрЃарЃўрЃЌрЃЏрЃћрЃЉрЃўрЃАрЃљ рЃЊрЃљ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃАрЃбрЃарЃБрЃЦрЃбрЃБрЃарЃћрЃЉрЃўрЃА рЃЋрЃўрЃЊрЃћрЃЮ рЃЏрЃљрЃАрЃљрЃџрЃљрЃќрЃћ, рЃАрЃљрЃЊрЃљрЃф рЃњрЃљрЃЋрЃўрЃЋрЃџрЃўрЃЌ leetcode-рЃўрЃА рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃћрЃЉрЃА, рЃарЃЮрЃЏрЃџрЃћрЃЉрЃўрЃф рЃЊрЃљрЃњрЃћрЃ«рЃЏрЃљрЃарЃћрЃЉрЃљ рЃбрЃћрЃЦрЃюрЃўрЃЎрЃБрЃарЃў рЃўрЃюрЃбрЃћрЃарЃЋрЃўрЃБрЃћрЃЉрЃўрЃА рЃњрЃљрЃЋрЃџрЃљрЃерЃў.

### Python - рЃўрЃА рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃА - 5 рЃћрЃбрЃљрЃърЃўрЃљрЃюрЃў рЃАрЃљрЃАрЃгрЃљрЃЋрЃџрЃЮ рЃњрЃћрЃњрЃЏрЃљ
1. I рЃћрЃбрЃљрЃърЃў - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃљрЃерЃў рЃЊрЃљ Python рЃАрЃљрЃцрЃБрЃФрЃЋрЃџрЃћрЃЉрЃў - рЃљрЃЦ рЃерЃћрЃю рЃерЃћрЃўрЃАрЃгрЃљрЃЋрЃџрЃў Python-рЃА, рЃарЃЮрЃњрЃЮрЃарЃф рЃърЃарЃЮрЃњрЃарЃљрЃЏрЃўрЃарЃћрЃЉрЃўрЃА рЃћрЃюрЃљрЃА, рЃњрЃљрЃљрЃЎрЃћрЃЌрЃћрЃЉ 50+ рЃЊрЃљрЃЋрЃљрЃџрЃћрЃЉрЃљрЃАрЃљ рЃЊрЃљ рЃърЃарЃЮрЃћрЃЦрЃбрЃА.
1. II рЃћрЃбрЃљрЃърЃў - рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў рЃЋрЃћрЃЉ рЃбрЃћрЃЦрЃюрЃЮрЃџрЃЮрЃњрЃўрЃћрЃЉрЃерЃў, рЃерЃћрЃАрЃљрЃЋрЃљрЃџрЃў Backend-рЃерЃў - рЃљрЃЦ рЃгрЃљрЃарЃЏрЃЮрЃЊрЃњрЃћрЃюрЃљ рЃерЃћрЃњрЃћрЃЦрЃЏрЃюрЃћрЃЉрЃљ рЃЊрЃўрЃЊ рЃАрЃБрЃарЃљрЃЌрЃќрЃћ, рЃњрЃљрЃўрЃњрЃћрЃЉ рЃЌрЃБ рЃарЃљрЃА рЃљрЃЎрЃћрЃЌрЃћрЃЉрЃА backend рЃЊрЃћрЃЋрЃћрЃџрЃЮрЃърЃћрЃарЃў, рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ рЃЏрЃЮрЃюрЃљрЃфрЃћрЃЏрЃЌрЃљ рЃЉрЃљрЃќрЃћрЃЉрЃА, рЃЊрЃљрЃЋрЃўрЃгрЃДрЃћрЃЉрЃЌ Django-рЃА рЃАрЃгрЃљрЃЋрЃџрЃљрЃА.
1. III рЃћрЃбрЃљрЃърЃў - Django Rest Framework - рЃЋрЃўрЃАрЃгрЃљрЃЋрЃџрЃўрЃЌ API рЃљрЃгрЃДрЃЮрЃЉрЃљрЃА DRF рЃАрЃљрЃерЃБрЃљрЃџрЃћрЃЉрЃўрЃЌ, рЃњрЃљрЃЏрЃЮрЃЋрЃДрЃЮрЃцрЃЌ рЃЎрЃџрЃўрЃћрЃюрЃбрЃА рЃАрЃћрЃарЃЋрЃћрЃарЃўрЃАрЃњрЃљрЃю, рЃЋрЃўрЃЏрЃБрЃерЃљрЃЋрЃћрЃЉрЃЌ рЃЊрЃљрЃЏрЃљрЃбрЃћрЃЉрЃўрЃЌ рЃЋрЃћрЃЉ рЃърЃарЃЮрЃарЃћрЃЦрЃбрЃћрЃЉрЃќрЃћ.
1.
